# 🚀 Chiara AI Consulting – DevOps Project

This project demonstrates a full end-to-end DevOps workflow by deploying a React-based AI consulting website using modern DevOps tools and practices.

---

## 🌐 Live Architecture (Planned)

GitHub → CI/CD (GitHub Actions) → Docker → Kubernetes (k3s) → ArgoCD → VPS  
                                                    ↓
                                             Monitoring (Prometheus + Grafana)

---

## 🧱 Tech Stack

- Frontend: React + TailwindCSS
- Containerization: Docker
- Orchestration: Kubernetes (k3s)
- GitOps: ArgoCD
- CI/CD: GitHub Actions
- Infrastructure: VPS (Ubuntu)
- Monitoring: Prometheus & Grafana

---

## 📦 Project Goals

- Containerize a React application
- Deploy using Kubernetes
- Implement GitOps with ArgoCD
- Automate deployments with CI/CD
- Monitor application and infrastructure
- Replace traditional hosting (Lovable/S3) with VPS-based deployment
